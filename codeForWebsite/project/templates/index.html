Overview

This software will create a predictive report based on machine learning approaches for pollutants in a place and time suggesting which crop is more resistant to the given data.

Parameters

For the given pollution data, which crop is best suited to withstand the pollutants, what will be the estimated rain for that crop, in what quantity the chemicals of the soil must be in, such that the crop can sustain more life, all these parameters are taken into consideration for building the model, which takes up these parameters and predicts the best crop that shall be grown.
